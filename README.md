A Progressive Web App (PWA) for an immersive museum experience with AR, interactive maps, and audio/video guides.

---

## 🔥 Features

- Augmented Reality (AR) integration for artworks
- Interactive museum maps and guides
- Audio and video guides for exhibits
- Admin dashboard for museum management
- Progressive Web App with offline support

---

## 💻 Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS, Vite  
- **Backend:** Node.js, Express, MongoDB  
- **Deployment:** Docker, Nginx  

---

## 🚀 Installation & Usage 

1. Clone the repository:
   git clone https://github.com/kajalkamble0713-jpg/RealMeta-Museum-PWA.git

2.Navigate into the project folder:
cd "RealMeta Museum PWA"

3.Install frontend dependencies and start frontend:
npm install
npm start

4.Setup backend:
cd backend
npm install
npm run dev

5.Open your browser at http://localhost:3000
